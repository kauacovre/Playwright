Video Summarizer em Python

Projeto em Python para capturar, processar e resumir vídeos automaticamente utilizando OpenCV para manipulação de vídeo e Playwright para automação web.

A ferramenta permite automatizar a obtenção de vídeos, processar frames e gerar resumos ou análises do conteúdo, facilitando workflows de análise de mídia ou automação de conteúdo.

Funcionalidades:
- Automação de captura ou acesso a vídeos via Playwright
- Processamento de vídeo utilizando OpenCV
- Extração de frames relevantes
- Base para construção de sistemas de resumo automático de vídeos
- Implementado totalmente em Python

Tecnologias Utilizadas:
- Python 3.10+
- OpenCV — processamento de vídeo e imagens
- Playwright — automação de navegador
  Pip / Virtualenv — gerenciamento de dependências

Fluxo esperado:
1. O Playwright acessa ou captura o vídeo
2. O OpenCV processa os frames
3. O script analisa e gera um resumo ou saída processada
